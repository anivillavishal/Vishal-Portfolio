# Vishal Anivilla Portfolio - Project Context & Memory

This is the central knowledge brain for the Antigravity agent when operating in this repository.

## 🎬 Project Context & Aesthetic 
This is a High-Cinematic Designer Portfolio for Vishal Anivilla, focusing on Film & Digital Content Production. It is built using **Next.js (App Router), React, Tailwind CSS, and Framer Motion**.

The design language is **"Darkroom"**:
- **Palette:** Deep pure blacks `#000000`, stark whites, heavy cinematic red accents `#FF2A00`.
- **Motion:** Complex GSAP/Framer animations, horizontal/vertical scroll pinning, cinematic loaders, glassmorphism UI, and smooth/parallax scrolling.
- **3D Assets:** Rotating abstract camera lens effects using `@react-three/fiber` and `@react-three/drei`.
